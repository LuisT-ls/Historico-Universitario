Política de Privacidade

Última atualização: 12 de Abril de 2025

1. Armazenamento Local de Dados

O Histórico Universitário opera com total privacidade através do armazenamento local:

  • Todos os dados são armazenados exclusivamente no localStorage do seu navegador
  • Nenhum dado pessoal ou acadêmico é transmitido pela internet
  • Não utilizamos servidores para armazenamento de dados
  • Não mantemos banco de dados dos usuários

2. Dados Armazenados Localmente

Os seguintes dados são salvos apenas no seu dispositivo:

  • Informações das disciplinas cursadas
  • Notas e carga horária
  • Preferências do usuário
  • Configurações da aplicação

3. Persistência e Remoção dos Dados

Sobre o armazenamento e remoção de dados:

  • Os dados permanecem salvos até que você os remova
  • Para remover todos os seus dados, você pode:
    • Utilizar o botão "Limpar Histórico" na interface principal, que após confirmação excluirá todas as disciplinas cadastradas
    • Limpar manualmente o localStorage do seu navegador através das configurações do navegador
  • Recarregar a página ou usar a tecla Shift não apaga os dados
  • Navegação anônima/privativa não mantém os dados após fechar o navegador

4. Backup e Portabilidade de Dados

O sistema oferece opções para garantir a portabilidade de seus dados:

  • Você pode exportar seus dados usando o botão "Exportar Histórico" disponível na interface
  • Formatos de exportação incluem PDF, TXT e CSV/Excel
  • Recomendamos fazer backup dos dados regularmente antes de realizar a limpeza do histórico

5. Recomendações de Privacidade

Para proteger seus dados, recomendamos:

  • Usar apenas em dispositivos pessoais e confiáveis
  • Não acessar em computadores públicos ou compartilhados
  • Utilizar o botão "Limpar Histórico" ao finalizar o uso em dispositivos compartilhados
  • Fazer backup regular dos seus dados usando a opção de exportação

6. Armazenamento em Cache e Análise

Informações sobre cache e rastreamento:

  • Utilizamos Service Worker para armazenar arquivos em cache, permitindo que o aplicativo funcione offline e carregue mais rapidamente
  • O cache armazena apenas arquivos da aplicação (HTML, CSS, JavaScript, imagens), não seus dados acadêmicos
  • Utilizamos Google Analytics apenas para análise anônima de uso
  • Não utilizamos cookies para rastreamento de usuários
  • Não coletamos dados de identificação pessoal
  • Não compartilhamos informações com terceiros
