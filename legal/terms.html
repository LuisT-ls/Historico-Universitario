Termos de Uso

Última atualização: 12 de Abril de 2025

1. Aceitação dos Termos

Ao acessar e usar o Histórico Universitário, você concorda com estes termos de uso. Se você não concordar com qualquer parte destes termos, não deverá usar o serviço.

2. Descrição do Serviço

O Histórico Universitário é uma ferramenta local para auxiliar estudantes no acompanhamento de seu progresso acadêmico. O serviço funciona inteiramente no seu navegador e todos os dados são armazenados localmente no seu dispositivo.

3. Armazenamento de Dados

É importante entender que:

  • Todos os dados são armazenados exclusivamente no localStorage do seu navegador
  • Nenhuma informação é enviada ou armazenada em servidores externos
  • Os dados persistem mesmo após fechar o navegador ou reiniciar o computador
  • É recomendado utilizar o serviço em um computador pessoal para maior privacidade
  • Para remover seus dados, você pode utilizar o botão "Limpar Histórico" disponível na interface do aplicativo, que removerá todas as disciplinas cadastradas após confirmação
  • Alternativamente, você pode limpar manualmente o localStorage do navegador para remover todos os dados do aplicativo

4. Recomendações de Uso

Para melhor experiência e segurança:

  • Utilize um computador pessoal, não um dispositivo compartilhado
  • Mantenha seu navegador atualizado
  • Faça backups regulares de seus dados exportando-os usando o botão "Exportar Histórico"
  • Não utilize o modo de navegação anônima/privativa, pois os dados não serão salvos

5. Limitação de Responsabilidade

Não nos responsabilizamos por:

  • Perda de dados após confirmação de exclusão pelo botão "Limpar Histórico"
  • Perda de dados devido à limpeza do localStorage do navegador
  • Acesso não autorizado aos dados em dispositivos compartilhados
  • Problemas técnicos relacionados ao navegador ou dispositivo do usuário
  • Precisão dos dados inseridos pelo usuário
